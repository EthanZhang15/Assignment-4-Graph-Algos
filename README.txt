To run the code, simply execute 'make run' in the command line. 
This will compile our code (graphAlgos.cpp) and execute it against
all 3 of the dimac files. The dimac formatted CSR representation
will be outputted in the respective 'name + "Output".dimacs' file,
and the node labels in the respecttive 'name + "NodeLabelOutput".txt'
file.
